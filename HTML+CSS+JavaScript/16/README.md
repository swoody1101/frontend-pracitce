    <h4>객체 알아보기</h4>
    <ul>
      <li><b>객체(object): </b>프로그램에서 인식할 수 있는 모든 대상</li>
      <li><b>프로퍼티(property): </b>객체의 특징이나 속성</li>
      <li><b>메서드(method): </b>객체에서 할 수 있는 동작</li>
      <li>객체 인스턴스 만들기</li>
      <pre><code>
      new 객체명
      </code></pre>
      <li>객체의 프로퍼티와 메서드 사용하기</li>
      <pre><code>
      객체명.프로퍼티
      객체명.메서드
      </code></pre>
    </ul>
